FROGZ

Welcome to Frogz! Frogz is a micro-hosting website that aims to help everyone create their own internet presence.

You can easily create your own pages using plain text, MarkDown or HTML. No registration required.

Click here to create a page!

Features

  • Absolutely FREE and with no adds.
  • No coding knowledge required.
  • Totally anonymous - you don't even need to register an account.
  • Pages are editable with a defined password.
  • Virtually no censorship - we only remove pages in very specific cases.
